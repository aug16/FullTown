WHAT

什麼是複合木

複合木是業界唯一含有65%實木木粉的複合式材料，不需額外的表面處理，也沒有使用膠合劑貼合外皮，原料木塑粒子經由高溫擠壓一體成型，堅固耐用，不同於一般塑膠仿木的產品。
複合木擁有極高的實木質感且耐用，非金屬烤木紋及其他塑木能達到的質感。
不論是屋頂、外牆、室內牆面、天花板、地板、格柵、欄杆扶手等，都可提供您不同造型及因應您所有需求的全方位選擇。

ASTM E84
  • ASTM E84 建築材料表面燃燒特性的測試方法
  • ASTM E84 Standard test method for surface burning characteristics of building materials。
  • 火焰傳播是指火焰在材料表面的發展，它關係到火災信鄰近可燃物而使火勢擴大，火焰傳播性能常以隧道法及輻射板法測試。 此ASTM E84法用於測定建築材料的火焰傳播速率(同時測定煙霧濃度)，按ASTM E84的規定，ASTM E84隧道法的所用設備為一個長為7.62m，開口端橫截面為0.45m*0.30m的內襯耐火磚的鋼槽，槽側有窗口。
  • 樣品至少1個，尺寸為0.51m*7.32m*使用最大厚度。點燃源為2個煤氣噴燈，能量輸出為5.3MJ/min，位於試件之下，距樣品190mm，平行於試驗室火的末端，相距305mm，根據試驗測得的火焰傳播速率值Flame Spread Index (FSI值)及煙密度值將材料分類。
  • 火焰傳播速率值FSI值越小的材料，火災危險性越小。高層建築和樓道，應採用FSI＜25的材料，25＜FSI＜100的材料只能用於防火要求不是很嚴格的場所，而FSI＞100的材料不符合阻燃的要求。
  • ASTM E84防火等級劃分
    • ASTM E84 Class A為0-25且煙指數不超過450。
    • ASTM E84 Class B為26-75且煙指數不超過450。
    • ASTM E84 Class C類76-200且煙指數不超過450。
